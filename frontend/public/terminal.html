Comment se connecter à ce terminal
1
Le superviseur ouvre SurveillancePage sur son poste (ordinateur connecté à internet).
2
Ce terminal s'ouvre sur le poste candidat via l'URL :
https://na2quizapp.netlify.app/terminal.html
3
Le superviseur sélectionne une épreuve et clique Distribuer. Ce terminal reçoit l'épreuve et redirige automatiquement le candidat vers le formulaire de profil.
4
Le candidat remplit son profil, rejoint la salle d'attente, puis compose selon l'option choisie (A / B / C / D).
5
Le bulletin est accessible après soumission via /api/bulletin/:id — imprimable depuis le navigateur.
